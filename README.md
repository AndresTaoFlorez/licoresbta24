# Licores BTA24

Sitio web comercial para venta de licores a domicilio en Bogotá. Catálogo de productos con servicio 24 horas.

## Características

- Interfaz responsive
- Catálogo de productos
- Integración con Google Ads

## Tecnologías

- React + Vite
- SCSS
- React Router
